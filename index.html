04 — Account Credentials
Bluesky
Go to Bluesky Settings → App Passwords → Add App Password.
Use that password here, NOT your main login password.
Facebook (+ Instagram mirror)
You need a Facebook Developer account (free).
1. Go to developers.facebook.com → Create App → Business type
2. Add "Pages" product → generate a Page Access Token
3. Make sure your Facebook Page has Instagram linked under Settings → Linked Accounts
Mastodon
In Mastodon: Settings → Development → New Application
Give it read+write permissions, copy the Access Token.
YouTube
Not connected
1. Go to console.cloud.google.com → New Project
2. APIs & Services → Enable APIs → search "YouTube Data API v3" → Enable
3. Credentials → Create OAuth Client ID → Web application
4. Under "Authorized JavaScript origins" add: null
5. Paste the Client ID above, then click Connect.
